## Notes

- MPEG-2 descriptors (0x02–0x12) are defined in ISO/IEC 13818-1 but appear in DVB PMTs
- DVB descriptors start at 0x40 (§6.2 of this spec)
- Extension descriptor (0x7F) is dispatched by `descriptor_tag_extension` field (§6.2.16, §6.4)
- Descriptor placement constraints: see spec §6.1 Table 12 for which descriptors are allowed in each table
