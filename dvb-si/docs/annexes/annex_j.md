# Annex J — Supplementary Audio Signalling

**Spec:** ETSI EN 300 468 v1.19.1 Annex J (normative)

Defines how `supplementary_audio_descriptor` (extension tag 0x06) is used to
signal audio description, hearing-impaired, and receiver-mix audio tracks.
