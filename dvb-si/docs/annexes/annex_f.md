# Annex F — ISO 639 Language Descriptor for Original Audio

**Spec:** ETSI EN 300 468 v1.19.1 Annex F (informative)

Informative guidance on using `ISO_639_language_descriptor` (tag 0x0A) to signal
the original audio language track. Relevant for multi-audio services.
